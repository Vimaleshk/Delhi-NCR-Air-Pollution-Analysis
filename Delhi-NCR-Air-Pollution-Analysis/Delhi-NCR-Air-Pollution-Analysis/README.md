# Delhi NCR Air Pollution Analysis

##  Overview
This project presents an **Exploratory Data Analysis (EDA)** of air pollution trends in the **Delhi NCR region** using Python. The goal is to uncover seasonal, hourly, and pollutant-specific patterns that highlight the environmental challenges faced by one of India’s most urbanized zones.

##  Objectives
- Clean and preprocess raw air quality datasets
- Perform statistical analysis and identify correlations between pollutants
- Visualize seasonal and temporal pollution trends
- Derive insights into peak pollution periods and pollutant dominance

##  Tech Stack
- **Python**
- **Libraries**: Pandas, NumPy, Matplotlib, Seaborn

##  Key Insights
- Seasonal spikes in pollution during **winter months** due to stubble burning and weather conditions
- **PM2.5 and PM10** identified as dominant pollutants
- Hourly variations show higher pollution levels during **morning and evening traffic hours**
- Correlation analysis highlights relationships between pollutants and meteorological factors
